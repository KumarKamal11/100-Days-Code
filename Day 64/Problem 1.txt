Day 64: Problem-1

You are given an integer array pref of size n.
Find and return the array arr of size n that satisfies:
pref[i] = arr[0] ^ arr[1] ^ ... ^ arr[i].
Note that ^ denotes the bitwise-xor operation.
It can be proven that the answer is unique.

Problem Link: https://leetcode.com/problems/find-the-original-array-of-prefix-xor/description/

Approach

Xor property:
b = a^c => a=b^c;
Since pref[i] = pref[i-1]^array[i] ,
therefore we can use, array[i] = pref[i]^pre[i-1]
where array[i] is ith element in desired array

Code:
       vector<int> ans;
        res.push_back(pref[0]);
        for (int i = 1; i < pref.size(); i++)
        {
            ans.push_back(pref[i] ^ pref[i - 1]);
        }
        return ans;
    }
